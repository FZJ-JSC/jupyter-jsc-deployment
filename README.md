# Jupyter-JSC Deployment
These files are used to deploy the following webservices at [JSC](https://www.fz-juelich.de/en/ias/jsc):  
 - [Jupyter-JSC](https://jupyter-jsc.fz-juelich.de)  
 - [JUNIQ](https://juniq.fz-juelich.de)  
 - [Jupyter CoE Raise](https://jupyter.coe-raise.eu)  
 - [Portal Gauss Centre](https://portal.gauss-centre.eu)  
 - [Jupyter EuroCC](https://jupyter.eurocc-gcs.de)  
 - [nbviewer for documentation](https://docs.jupyter-jsc.fz-juelich.de)  
  
Belonging base services can be found at this [branch](https://github.com/FZJ-JSC/jupyter-jsc-deployment/tree/basics-production).
  
We're using [fleet](https://fleet.rancher.io/) to deploy this repository to our clusters.  
  
Our K8s clusters are running on the [HDF-Cloud](https://www.fz-juelich.de/en/ias/jsc/systems/scientific-clouds/hdf-cloud).
